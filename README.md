# Palindrome Checker

A Python function that determines whether a given string is a palindrome.
Spaces, punctuation, and letter case are all ignored during the check.

## What It Does
- Strips all non-letter characters and converts to lowercase before checking
- Returns True if the cleaned string reads the same forwards and backwards
- Returns False for non-string inputs

## Sample Output
is_palindrome("Nasa bayabasan")       # True
is_palindrome("O! tatayo yata 'to.")  # True
is_palindrome("Ghani")                # False

## Requirements
- Python 3.x
